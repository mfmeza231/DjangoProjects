IoT Meme Project

This is the classic AWS Meme Generator project that can be found at https://acloud.guru/series/acg-projects/view/104. The project is designed to post a meme to a Slack chat everytime an AWS IoT Dashbutton is pressed. This project instead posts the meme here to my website. The additional services required include AWS IoT, Lambda and Kinesis. AWS IoT is used to register the AWS Dashbutton to my AWS account. A Lambda function was created to handle the Dashbutton click event which then generates a meme. Finally, Kinesis handles routing the meme to my PosgreSQL Database in RDS.
